### 1. Sou a Joana

![Sou a Joana Portafolio](https://github.com/rosi2701/trabajo-final/blob/main/img_yesyes/joana.png?raw=true)

Qué es: Portafolio de una diseñadora gráfica y dirección de arte portuguesa.
Fuente: <https://souajoana.com/>

Aspectos positivos:
  - La página principal presenta una identidad visual muy marcada
  - Sitio web intuitivo

Aspectos negativos:
  - Las páginas que muestran los proyectos son muy simples
  - Ciertas interacciones son sutiles y pueden perderse

### 2. Yifei Luo

![Yifei Luo Portafolio](https://github.com/rosi2701/trabajo-final/blob/main/img_yesyes/yifeiluo.png?raw=true)

Qué es: Portafolio de una diseñadora gráfica y artista digital china.
Fuente: <https://luoooif.com/works>

Aspectos positivos:

Aspectos negativos:

### 3. Better Weather Studio

![Better Weather Studio](https://github.com/rosi2701/trabajo-final/blob/main/img_yesyes/betterweather.png?raw=true)

Qué es: Sitio web de un estudio de diseño danés que produce muebles y objetos.
Fuente: <https://betterweather.dk/>

Aspectos positivos:
  - Sitio web intuitivo
  - Buena manera de destacar los productos (destaca imágenes de producto con fondos neutros)

Aspectos negativos:
  - Desfase visual entre la tienda y la página principal
